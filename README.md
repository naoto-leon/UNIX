# UNIX
コマンドプロントUNIX記載　チートシート


### pwd
現在のディレクトリ  

### cd  
ディレクトリ移動  
(EX) cd Desktop  

### ls  
入っているファイル参照  
(EX) ls Desktop  

### cp-r xxx xxx2  
xxxをxxx2としてコピー  

### mkdir  
新規ファイルを作る  
mkdir -p xxx/xxx2 とした場合xxxも合わせて作られる  

### mv xxx xxx2  
ファイル移動　xxx2をxxxに入れる  

### rmdir  
ファイルの削除  

### rm -r xxx  
xxxの中含めて全て削除。  

### less  
テキストとかの中身見る  
q で閉じる  


## Vim 
編集tool

### vi xxx  
xxxをvimで開く
