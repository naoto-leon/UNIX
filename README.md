# UNIX
コマンドプロントUNIX記載　チートシート


### pwd
現在のディレクトリ  

### cd  
ディレクトリ移動  
(EX) cd Desktop  

### ls  
入っているファイル参照  
(EX) ls Desktop  

### cp-r xxx xxx2  
xxxをxxx2としてコピー  

### mkdir  
新規ファイルを作る  
mkdir -p xxx/xxx2 とした場合xxxも合わせて作られる  

### mv xxx xxx2  
ファイル移動　xxx2をxxxに入れる  

### rmdir  
ファイルの削除  

### rm -r xxx  
xxxの中含めて全て削除。  

### less  
テキストとかの中身見る  
q で閉じる  

### $  
引数の引き渡し  
(EX)  
ls xxx  
cd !$  
!$ == xxx  

***

## Vim 
編集tool

### vi xxx  
xxxをvimで開く

### i  
insertの開始  

### ESC  
編集終了  

### :w  
保存  

### :q  
閉じる  

### :q!  
保存せずに終了
